# Digital Project: AI-Driven Diagnostics in Modern Healthcare

A modern, single-page website presenting research on the role of artificial intelligence in medical diagnostics.

## Features

- **Modern Design**: Built with Tailwind CSS for a sleek, professional look
- **Poppins Font**: Clean, readable typography from Google Fonts
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop
- **Visual Elements**: Custom SVG graphics and gradient designs
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **MLA References**: Properly formatted citations
- **Easy Deployment**: Ready for Vercel with zero configuration

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Deployment to Vercel

### Option 1: Using Vercel CLI

1. **Install Vercel CLI** (if not already installed):
   ```bash
   npm install -g vercel
   ```

2. **Login to Vercel**:
   ```bash
   vercel login
   ```

3. **Deploy**:
   ```bash
   vercel
   ```

### Option 2: Using GitHub (Recommended)

1. Push your code to a GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Vercel will automatically detect and deploy your site

## Local Development

Simply open `index.html` in your web browser. No build process required!

## Customization

- **Content**: Edit text directly in `index.html`
- **Colors**: Modify Tailwind classes (e.g., `from-blue-600` to `from-green-600`)
- **Layout**: Adjust grid and spacing using Tailwind utility classes
